microHarnesses

A composable, plugin-first library for building LLM agent harnesses.

The idea is simple: keep a small runtime core, then add only the capabilities you need. Instead of a fixed framework, microHarnesses lets you compose providers, tools, policy, planning, and subagents as independent modules.

How the library is structured

@micro-harness/core Runtime loop, tool execution, context/session management, plugin host.
@micro-harness/providers Provider adapters and credentials resolvers (OpenAI, Anthropic, Ollama).
Plugin packages Add focused capabilities like planning mode, subagents, and utility tools without bloating core.

Why plugins

Plugins keep the system flexible: teams can swap implementations, enforce policy, and extend behavior without forking the runtime. You build the agent stack you actually need, and evolve it incrementally.
